# 1. Instructional Design and Curriculum Planning

These tools are used to design **learning pathways, course structures, and instructional frameworks**.

## Core Tools

| Tool | Purpose |
|-----|------|
| **Miro** | Visual curriculum mapping and learning flow diagrams |
| **Figma** | Learning experience design and instructional prototypes |
| **Notion** | Course documentation, module planning, knowledge management |
| **Lucidchart** | Technical diagrams and process flow mapping |
| **Whimsical** | Concept maps and visual learning models |

## Use Case

Technical instructors use these tools to create:

- course blueprints
- concept maps
- learning journeys
- system architecture diagrams
- engineering workflows

# 2. E-Learning Authoring Tools

Authoring tools are used to create **interactive courses, quizzes, simulations, and digital lessons**.

## Industry Standard Tools

| Tool | Function |
|-----|------|
| **Articulate 360** | Interactive course development |
| **Articulate Storyline** | Scenario-based learning modules |
| **Adobe Captivate** | Software simulations and interactive learning |
| **iSpring Suite** | Rapid e-learning development |
| **Easygenerator** | Fast course authoring |
| **SmartBuilder** | Complex interactive training programs |

## Output Formats

These tools typically export:

- SCORM packages
- xAPI learning objects
- HTML5 learning modules

These formats are compatible with most LMS systems.
